2) Для того, чтобы удалить пользователя нужно испоьзовать утилиту userdel:
sudo userdel//sudo userdel -r(Удаление вместе с директорией)
	linyx3@ubuntu:~$ sudo userdel ilya
	[sudo] password for linyx3: 

3)Чтобы создавть группу для пользователя нужно:
	sudo groupadd new_group

4) Чтобы добавить текущую учетную запись в группу нужно:
	usermod -a -G group_name user_name(Этой командой мы меняем название группы на свое, здесь - user_name)

5) Чтобы добавить пользователя в группу:
	usermod -a -G grupka user(Мы пользователя user добавляем в группу grupka)

6) Чтобы узнать в какие группы входит пользователь вводим команду
	groups

7)мы уже смогли добавить пользователя  сгруппой в еще одну группу, но теперь мы поменяем первичную группу пользователя:
	usermod -g group_name user_name(Когда вы используете маленькую g, вы назначаете главную группу.)

8) УДАЛЕНИЕ ПОЛЬЗОВАТЕЛЯ ИЗ ГРУППЫ LINUX:
Для этого мы должны использовать команду 
	gpasswd -d user grupka//(Опция -d сообщает, что нужно выполнить удаление)
